Accessibility Statement

Our commitment to making this site usable by everyone.

Our commitment

Good Dog Roasters is committed to ensuring that our website is accessible to people with disabilities. We aim to meet the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA — the standard referenced by the Americans with Disabilities Act (ADA) for website accessibility.

What we've done

We have taken the following steps to improve accessibility on this site:

  • Semantic HTML landmarks so screen readers can navigate the page structure (main, nav, footer)
  • Skip-to-content link available on keyboard focus
  • All images have descriptive alternative text
  • All form inputs have associated labels
  • Color contrast meets or exceeds the WCAG 2.1 AA minimum ratio of 4.5:1 for normal text
  • All interactive elements (buttons, links) are keyboard navigable with visible focus indicators
  • Background video includes a pause/play control
  • Animations respect the prefers-reduced-motion user setting
  • Heading levels follow a logical hierarchy (h1 → h2 → h3)
  • The page language is declared (lang="en")

Known limitations

We are continuously working to improve accessibility. Current known items in progress:

  • The hero background video does not include audio description (the video is decorative and contains no dialogue or meaningful audio)
  • An Open Graph preview image for social sharing is pending

Assistive technology compatibility

This site is designed to be compatible with the following assistive technologies:

  • Screen readers: NVDA, JAWS, VoiceOver (macOS and iOS), TalkBack (Android)
  • Keyboard-only navigation
  • Browser zoom up to 200%
  • High contrast mode

Contact us about accessibility

If you experience any difficulty accessing content on this site, or if you find that something is not working as expected with your assistive technology, please contact us. We take all accessibility feedback seriously and will work to resolve issues promptly.
