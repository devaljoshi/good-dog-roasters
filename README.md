# Good Dog Roasters

Artisan coffee brand website. Hybrid model: limited-drop artisan roasts (via Wait.li on Shopify) + everyday a la carte and subscription coffees on Shopify.

## Files

- `index.html` — Main site (hero, story, products, waitlist, footer)
- `privacy.html` — Privacy policy
- `good_dog_roasters_oval_standard.png` — Logo
- `videos/header.mp4` — Hero parallax video (add this folder with your video file)

## Setup Instructions

1. Add `header.mp4` to a `/videos` folder in the repo
2. Push to GitHub
3. Connect to Netlify via GitHub
4. Point `good-dog-roasters.com` domain to Netlify

## Shopify Integration

- Main store: `https://good-dog-roasters.myshopify.com`
- Collections: Single Origin, Blends, Flavored, Sample Packs, Tea, Merch
- Wait.li app installed for drop-list access to artisan roasts

## Brand

- Tagline: Be Good. Do Good. Taste Good.
- Palette: #fdbb56 (gold), #799b8e (sage), #d4bfaa (sand), #f6f3f1 (cream)
- Fonts: Playfair Display (display) + DM Sans (body)
- Location: South Florida
